A simple API based plugin for fetching data from miusage.

== Description ==
Fetch and display data from miusage api.

This api data refresh every hour.

= Contribute =
If you find bugs, plase make issues on Github. Any pull requests are welcomed.

= Author =
Created by [hasinur1997](https://github.com/hasinur1997)

== Installation ==

Extract the zip file and just drop the contents in the wp-content/plugins/ directory of your WordPress installation and then activate the Plugin from Plugins page.
